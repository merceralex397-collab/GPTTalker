### AC3: 401 returned when auth configured but missing/invalid ✓ PASS

**Evidence from `src/node_agent/dependencies.py` lines 37-42:**
```python
auth_header = request.headers.get("Authorization", "")
if not auth_header.startswith("Bearer "):
    raise HTTPException(status_code=401, detail="Missing or invalid Authorization header")

token = auth_header[7:]  # Strip "Bearer " prefix
if token != config.api_key:
    raise HTTPException(status_code=401, detail="Invalid API key")
```

Two failure cases produce 401:
1. Missing or malformed `Authorization` header (not starting with "Bearer ")
2. Token present but doesn't match configured `api_key`
