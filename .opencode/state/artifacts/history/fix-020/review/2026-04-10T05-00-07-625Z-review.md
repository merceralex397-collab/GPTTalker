# Code Review: FIX-020 — Defect 5 (MCP error double-wrapping)

## Ticket
- ID: FIX-020
- Title: Fix missing authentication enforcement on node agent operational routes
- Stage: review
- Kind: review

## Defect 5 Scope

**File changed:** `src/hub/transport/mcp.py`

**Bug:** `format_tool_response` was wrapping handler-returned error dicts (`{"success": False, "error": "..."}`) inside `result.data`, producing malformed JSON-RPC:
```json
{"jsonrpc": "2.0", "id": ..., "result": {"success": false, "data": {"success": false, "error": "..."}}}
```

**Fix:** Added error-promotion check before the success-path handling at line 138.
