# FIX-024: Fix node-client response envelope stripping and path-mode search output parsing

## Summary

Fix two post-completion defects found during FIX-020 final code review: (1) HubNodeClient read_file/search/git_status/list_directory strip the OperationResponse envelope and return only inner data, causing successful operations to appear as failures to MCP callers who expect success/error at top level. (2) Node executor path-mode search uses --files-with-matches but parses output as file:line:content, silently returning zero results.
