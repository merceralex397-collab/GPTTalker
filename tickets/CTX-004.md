# CTX-004: Issue tracking tools

## Summary

Implement MCP tools for tracking known issues across projects: list_known_issues for viewing issues and record_issue for creating new issue records. Issues are stored in SQLite with title, summary, severity, and evidence fields. Issues are also indexed in Qdrant for semantic retrieval in context bundles.
