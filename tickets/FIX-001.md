# FIX-001: Fix walrus operator syntax error in opencode.py

## Summary

The walrus operator on line 399 of src/hub/tools/opencode.py uses `if scheduler_input.allow_fallback and schedule_result := getattr(...)` which is invalid Python — assignment expressions cannot appear directly in boolean expressions without parentheses. This causes a SyntaxError that blocks hub startup through the register_all_tools() → register_llm_tools() → opencode import chain. Fix by wrapping the walrus expression in parentheses.
