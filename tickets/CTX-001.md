# CTX-001: Qdrant integration

## Summary

Set up the Qdrant vector database integration for semantic search across project context. Implement client configuration, collection management with per-repo collections and a global summary collection, and the embedding pipeline that converts text into vectors via the embedding service adapter. This is the foundation for all context-aware features.
