# LLM-001: chat_llm tool

## Summary

Implement the chat_llm MCP tool that allows ChatGPT to route prompts to registered LLM services by alias. This is the primary interface for multi-model orchestration — ChatGPT can ask a coding agent to edit code, a general LLM to analyze text, or a helper model to classify content. Responses include backend identification and timing metadata.

## Notes

- This tool is a router, not an LLM — it forwards prompts to registered backends
- Session ID enables multi-turn conversations with stateful backends like OpenCode
- Consider adding a "dry run" mode that returns routing info without calling the backend
- Rate limiting per service should be considered for future work
