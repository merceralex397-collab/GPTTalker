# FIX-016: Security hardening — path validation and config safety

## Summary

Two security issues: (1) _validate_path in executor.py resolves and accepts absolute paths as long as they fall under allowed root, instead of rejecting absolute paths outright as required by AGENTS.md security rules. (2) Both HubConfig and NodeAgentConfig use Pydantic extra='allow' which silently accepts typos and unknown env vars. Change to extra='ignore' for safer configuration.
