# OBS-002: History query tools

## Summary

Implement MCP tools for querying the task history system: list_task_history with filtering, get_task_details for individual task records, and list_generated_docs for finding documents created through the hub. These tools give ChatGPT visibility into what operations have been performed and their outcomes.
