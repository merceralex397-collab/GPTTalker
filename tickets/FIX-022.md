# FIX-022: Fix HubNodeClient.read_file HTTP method and endpoint mismatch

## Summary

FIX-022: HubNodeClient.read_file in src/hub/services/node_client.py uses HTTP GET /files/read?path={path} with no offset/limit. The node agent only exposes POST /operations/read-file with ReadFileRequest body that accepts path, offset, and limit. This contract mismatch causes the hub's read_file tool to always fail. Fix by changing to POST /operations/read-file with JSON body.
