# Proposal — Repo Discovery and Registration

## Simple explanation

This is about how repos enter the system.

In plain terms:
- do repos get found automatically, or do you register them manually?

## Status

Proposal document.

## Options

### Option A — Manual registration
Advantages:
- safest
- clearest
- no accidental exposure

Downsides:
- more admin work

### Option B — Folder scan discovery
Advantages:
- convenient
- good for repo-heavy machines

Downsides:
- can pick up junk or private repos you did not mean to expose

### Option C — Assisted discovery
The system finds candidates, but you approve them.

Advantages:
- good balance
- still operator-controlled

Downsides:
- more UI/workflow complexity

## Recommendation

Recommended proposal: assisted discovery, with manual approval before exposure to ChatGPT.
